üöÄ Quick Start

1. Prerequisites

You'll need a free API key from one of these services:

  • Google Gemini (recommended): Get Free API Key ‚Üí
  • OpenAI: Get API Key ‚Üí

2. Installation Steps

  1. Clone the repository:
    git clone https://github.com/jhfnetboy/XGarden.git cd XGarden
  2. Install dependencies (requires pnpm):
    pnpm install
  3. Build the extension:
    pnpm run build
  4. Load in Chrome:
    • Open chrome://extensions/
    • Enable "Developer mode" (top right)
    • Click "Load unpacked"
    • Select the dist folder

3. First Time Setup

  1. Click the XGarden extension icon in Chrome
  2. Go to Settings ‚Üí AI Config
  3. Enter your API key and select your preferred model
  4. Click "Create New World" and start building!

4. Create Your First World

  1. Name your world and set a description
  2. Add characters with unique personas
  3. Configure background images and music
  4. Start chatting with your AI characters!
