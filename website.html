üöÄ Quick Start

1. Prerequisites

You'll need a free API key from:

  • Google Gemini (recommended): Get API Key
  • OpenAI: Get API Key

2. Installation Steps

  1. Clone the repository:
    git clone https://github.com/jhfnetboy/XGarden.git cd XGarden
  2. Install dependencies:
    pnpm install
  3. Build the extension:
    pnpm run build
  4. Load in Chrome:
    • Open chrome://extensions/
    • Enable "Developer mode"
    • Click "Load unpacked"
    • Select the dist folder
  5. Configure your API key in XGarden Settings

3. Create Your First World

  1. Click the XGarden extension icon
  2. Click "Create New World"
  3. Add characters and define the world setting
  4. Start chatting with AI-powered characters!
